# 📝 To-Do List App

A simple and beginner-friendly To-Do List application built using HTML, CSS, and JavaScript.

## 🚀 Features

- Add new tasks
- Delete tasks
- Mark tasks as completed
- Edit tasks by double-clicking
- Data stored using Local Storage
- Simple and clean UI

## 🛠️ Technologies Used

- HTML5
- CSS
- JavaScript (DOM Manipulation & Local Storage)

## 💡 How It Works

- Users can type a task and click the Add button
- Tasks are displayed dynamically using JavaScript
- Completed tasks get a strike-through effect
- Tasks remain saved even after refreshing the page using Local Storage
